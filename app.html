Selamat datang ke Almaany API!

Almaany API adalah API untuk memperoleh data terjemah dari Bahasa Indonesia ke Bahasa Arab yang diambil dari situs Almaany.com

Anda telah terautentikasi!

URL untuk menggunakan API:

http://almaany-api.herokuapp.com/api?indo={kata bahasa indonesia}&token={masukkan token anda disini}

Contoh: http://almaany-api.herokuapp.com/api?indo=buku&token=*********
